Lord Of The Redirects

One Redirect to rule them all, One Redirect to find them, One Redirect to bring them all, and through the browser guide them.

You can use the following formats for URLs:

✅ Direct URLs

Use an exact URL to match and redirect a specific request.

Example:

  • Rule: https://example.com/script.js
  • Matches only this file: https://example.com/script.js

✅ Wildcard Matching (*)

Use wildcards to match multiple URLs with a common pattern.

Example:

  • Rule: https://*example.com/script.js*
  • Matches files like these:
    • https://www.example.com/script.js
    • https://example.com/script.js?v=123
